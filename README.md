# polkadot-api-tutorial

To install dependencies:

```bash
bun install
```

Adding Chains:
bunx papi add dot -n polkadot
bunx papi add people -n polkadot_people
bunx papi add collectives -n polkadot_collectives

To run:

```bash
bun run index.ts
```
